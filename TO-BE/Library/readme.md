# BPMN TO-BE схемы библиотеки

Репозиторий содержит улучшенные (TO-BE) BPMN-схемы бизнес-процессов университетской библиотеки.

## Состав
- Бронирование книги  
- Регистрация читателя  
- Выдача книги  
- Возврат книги  

## Описание
Схемы отражают целевое состояние процессов с учетом автоматизации: ускорение операций, снижение ошибок и контроль должников.

## Формат
Все диаграммы находятся в формате PNG
